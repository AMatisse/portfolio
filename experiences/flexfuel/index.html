Android Developer

FlexFuel

Rennes, Brittany, France

Mission

Sep 2019 - Jan 2020 (4 months)

Android Kotlin Bluetooth

Context

Joined a team of 3 developers to develop an Android application for FlexFuel Company, a leader in engine decarbonization solutions. The application aimed to analyze engine clogging via the OBD port and guide users through the data collection process. This solution was part of enhancing the engine decarbonization process, a proprietary technology by FlexFuel Company.

Responsibilities

Development and Optimization

  • Enhanced the engine clogging analysis system via the OBD port
  • Optimized application performance
  • Developed user interfaces
  • Implemented a data collection guide

Architecture and Design

  • Integrated with the OBD protocol
  • Designed algorithms for engine data analysis
  • Developed features using Kotlin

Organization and Methodology

  • Collaborated within a team of 3 developers
  • Participated in team meetings
  • Documented developed features
