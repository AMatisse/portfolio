Fullstack Developer

AluK

Rennes, Brittany, France

Mission

Feb 2020 - Feb 2021 (1 years)

Vue.js Node.js Sequelize PostgreSQL

Project

The primary objective of this project is to design a B2C commercial extranet for AluK, enabling the company’s sales team to track orders for all clients worldwide. This extranet aims to provide a robust and scalable platform to manage internationalization and specific needs related to global order management.

Context

AluK, a leader in aluminum joinery design, seeks to establish a centralized system for its sales team to monitor client orders globally. The development team, consisting of four developers, faces the challenge of handling the platform’s internationalization, with a focus on language management to ensure a consistent user experience across different regions of the world.

Responsibilities

Front-End Development

  • Implementation of multilingual features
  • Development of global search components
  • Management of international currencies
  • Creation of adaptive user interfaces

Back-End Development

  • Creation of new API routes
  • Integration with Sequelize for database management
  • Implementation of international order management
  • Configuration of the language management system

Maintenance and Optimization

  • Bug identification and resolution
  • Performance optimization
  • Technical documentation
  • Support for international users
