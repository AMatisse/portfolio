Albion Equipment

A comprehensive mobile utility application for Albion Online players that streamlines equipment management and provides essential game information. With over 45k downloads, it has become a trusted tool for the Albion community.

Features

  • Equipment Manager: Create, save, and share custom builds
  • Item Encyclopedia: Complete database with detailed information
  • Farming Timers: Track crop and animal breeding progress
  • Multi-language Support: Available in 9 languages

Technical Stack

  • Frontend: Vue.js with Vuetify
  • Mobile: Capacitor for native Android features
  • Backend: Docker containerized services
  • Database: Regular updates via game files parsing

Available Languages

Fully translated interface and database in:

  • English
  • French
  • German
  • Russian
  • Polish
  • Portuguese
  • Spanish
  • Italian
  • Bahasa Indonesia
